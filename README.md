# Mokap PEMEC

Aplicación de interfaz clínica para monitoreo y gestión de pacientes con enfermedades crónicas.

## Cómo ejecutar

1. Asegúrate de tener Node.js instalado (versión 16 o superior).
2. Instala las dependencias:
   ```
   npm install
   ```
3. Ejecuta el servidor de desarrollo:
   ```
   npm run dev
   ```
4. Abre tu navegador en `http://localhost:5173/` (o el puerto que indique la consola)

## Tecnologías utilizadas

- React 18
- Vite (bundler)
- Tailwind CSS (estilos)
- Lucide React (íconos)
- ESLint (linting)
- Vite
- Lucide React (íconos)
- Tailwind CSS (estilos incluidos en el componente)
